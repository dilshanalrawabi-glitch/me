# Muhammed Dilshan — Portfolio

A modern portfolio site built with **Next.js 15**, **TypeScript**, and **Tailwind CSS**.

## Tech stack

- **Next.js 15** (App Router)
- **React 19**
- **TypeScript**
- **Tailwind CSS**
- **clsx** (class names)

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Data to update

- **About** — Refine bio, education, and location in `src/components/About.tsx`
- **Experience** — Edit the `experience` array in `src/components/Experience.tsx`
- **Skills** — Adjust `skillGroups` in `src/components/Skills.tsx`
- **Projects** — Replace placeholder projects in `src/components/Projects.tsx` (titles, descriptions, tags, links)
- **Contact** — Set your real email, LinkedIn, and GitHub URLs in `src/components/Contact.tsx`
- **Metadata** — Update `title` and `description` in `src/app/layout.tsx` if needed

## Build

```bash
npm run build
npm start
```
